# File Allocation Table.

## Funcionamento.
Realiza a leitura de uma imagem em FAT16 e apresenta as seguintes informações:
- Os arquivos e subdiretórios na raiz.
- O conteúdo de um arquivo escolhido pelo usuário.
- O conteúdo de um subdiretório armazenado no diretório raiz.

## Imagens FAT.
Todos os testes foram feitos com o uso das imagens FAT que estão em anexo nesse repositório.
